TuApp Monetización 

App móvil en React Native + Firebase + MercadoPago para monetizar contenido digital con likes → créditos → dinero real. 

🚀 Instalación
1. Clonar el repositorio
   `bash
   git clone https://github.com/tuusuario/tuapp-monetizacion.git
   cd tuapp-monetizacion
   `
2. Instalar dependencias
   `bash
   npm install
   `
3. Configurar Firebase (Auth, Firestore, Storage).
4. Configurar MercadoPago (ACCESS_TOKEN).
5. Deploy del backend
   `bash
   firebase deploy --only functions
   `
6. Ejecutar la app
   `bash
   npx react-native run-android
   ` 

🔒 Seguridad
- Validación de identidad con documentos vigentes.
- Moderación automática de fotos/videos.
- Retiros seguros con MercadoPago.
`
